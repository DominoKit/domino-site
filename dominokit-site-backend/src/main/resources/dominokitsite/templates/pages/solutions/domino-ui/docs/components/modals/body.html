Basic modals Basic modal are popup cards that show on top of the page and can optionally mask the content.

Sheet modals Modal that appear attached to the edge of the screen.

Colored modal Different modal sections can have different colors.

Window modals Window modal can be dragged and moved inside the page and be maximized to full window mode.
