Domino UI

A type safe and feature rich UI components library for java developer without dependencies on external JavaScript.

Type Safe

Java based compiled to JavaScript, which means during development of the application you have the power to refactor and keep your app maintainable all the time.

Rich Components

Having rich interactive forms is essential for enterprise applications. We have a rich set of form controls to enable developers implement both simple and complex form easily.

Typography & Font Icons

Icons and Typography can enhance the user experience of the application. We have a variety of font icons that can bring forms and other elements in the page into life.
