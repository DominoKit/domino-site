Introduction

Domino-rest simplifies REST client creation from JaxRs-compatible interfaces, catering to both server and client-side (browser) needs. It seamlessly supports GWT2 and GWT3/J2CL environments, employing domino-jackson for serialization, JaxRs annotations for service definitions, and annotation processing (APT) for efficient code generation.

In a nutshell, Domino-rest is a versatile library that bridges the gap between client and server environments, delivering robust REST clients. By embracing industry standards and efficient tools like domino-jackson and annotation processing, it streamlines the process of generating clients for JaxRs interfaces, ensuring compatibility with various GWT environments.
