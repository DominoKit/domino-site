Request and Response mapping

The mappers for serialization and deserialization of the request body and response body in domino-rest from/to JSON will be automatically generated for POJOs, if the Request body bean or the response bean is annotated with @JSONMapper then domino-rest will use the generated mappers, otherwise it will generate new mappers for those beans.

for more about supported JSON types please check out domino-jackson

The JSON mapping will be applied only if the MediaType is for a json type, meanwhile if the MediaType is plain text type, domino-rest will use the string representation of body and response instead of JSON.

For other MediaTypes please check the Produces and Consumes section of this documentation.
